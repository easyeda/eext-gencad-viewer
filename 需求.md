# BOM器件智能封装匹配 — LCEDA扩展插件需求文档

## 一、项目概述

| 项目 | 说明 |
|------|------|
| 插件名称 | BOM器件智能封装匹配 |
| 插件定位 | EasyEDA Pro 编辑器内的器件选型与封装绑定工具 |
| 目标用户 | 需要将 OrCad / Altium Designer / PADS 等第三方EDA原理图迁移至 LCEDA 的硬件工程师 |
| 核心价值 | 将"手动搜索商城 → 查找封装 → 逐个绑定"的多步操作，简化为编辑器内一键匹配与批量绑定 |

## 二、需求背景

### 2.1 当前痛点

将第三方EDA软件的原理图文件转换至 LCEDA 后，存在以下问题：

1. 大部分元器件无法直接匹配 LCEDA 封装库；
2. 需手动打开立创商城/器件库搜索器件，再手动查找对应封装并完成绑定；
3. 操作流程繁琐、效率低下；
4. 易出现封装匹配错误、器件型号选错等问题。

### 2.2 期望效果

- 编辑器内一键查询器件，无需跳转外部页面；
- 自动匹配立创商城物料与 LCEDA 封装库；
- 支持批量绑定，大幅提升迁移效率；
- DRC 检查辅助校验，降低人为失误率。

## 三、整体业务流程

```
┌─────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌─────────────┐
│ 1. 上传BOM  │───▶│ 2. 提取数据类 │───▶│ 3. 查询匹配  │───▶│ 4. 展示结果  │───▶│ 5. 绑定封装 │
│    文件      │    │   选择依据    │    │  立创数据库   │    │  用户确认    │    │  参数同步   │
└─────────────┘    └──────────────┘    └──────────────┘    └──────────────┘    └─────────────┘
```

**详细步骤：**

1. 上传原始BOM文件（客户提供的BOM 或从源设计文件导出的BOM），插件自动提取BOM内的数据类；
2. 用户选择匹配依据（可选择一个或多个数据类作为匹配参数）；
3. 插件同步查询立创商城物料库与 LCEDA 官方器件库，获取匹配结果；
4. 在编辑器内展示器件匹配结果、封装预览图、器件详细参数；
5. 用户选择目标物料/器件后，点击一键绑定按钮，完成封装绑定；
6. 支持批量器件的批量匹配、批量绑定操作。

## 四、数据源定义

### 4.1 数据源

| 数据源 | 说明 | 提供内容 |
|--------|------|----------|
| 立创商城元件库 | 商城物料数据库 | 器件型号、参数、封装、库存、价格等完整信息 |
| LCEDA官方封装库 | EDA标准封装库 | 标准封装文件，确保封装兼容性 |

### 4.2 查询优先级

匹配立创数据库时，按以下顺序检索：

1. **基础库** — 优先匹配
2. **贴片库存** — 次优先
3. **拓展库** — 最后匹配

## 五、功能模块详细需求

### F1: BOM文件上传与解析

| 项目 | 说明 |
|------|------|
| 触发方式 | 插件打开后弹出上传页面 |
| 支持格式 | CSV、Excel（.xlsx/.xls）等常见BOM格式 |
| 解析逻辑 | 自动识别BOM表头，提取所有键值对参数列 |
| 输出 | 结构化的BOM数据表（含所有列字段与行数据） |

### F2: 数据类提取与匹配依据选择

| 项目 | 说明 |
|------|------|
| 数据类提取 | 自动从BOM表头提取所有数据类（如 Footprint、Value、Part Number、Description 等） |
| 展示方式 | 将所有数据类独立列出，以下拉/多选方式供用户选择 |
| 选择规则 | 用户可选择一个或多个数据类作为匹配依据 |
| 多依据匹配 | 选择多个数据类时，取交集进行精确匹配 |

### F3: 器件自动分组

| 项目 | 说明 |
|------|------|
| 分组规则 | BOM中所有键值对参数完全一致的器件归为一组 |
| 位号处理 | 同组器件的位号不做分组展示，收敛显示（如 "R1, R2, R3... 共12个"） |
| 目的 | 减少重复匹配操作，界面更简洁 |

### F4: 自动匹配与结果展示

| 项目 | 说明 |
|------|------|
| 匹配参数 | 以用户选择的数据类为匹配依据 |
| 匹配顺序 | 基础库 → 贴片库存 → 拓展库 |
| 结果展示 | 封装预览图 + 原理图符号 + 器件完整参数 |
| 多结果处理 | 匹配到多个结果时，列表展示供用户选择 |
| 可修改性 | 自动匹配结果可由用户人工修改和重新匹配 |

### F5: 手动匹配功能

| 项目 | 说明 |
|------|------|
| 适用场景 | 无法自动匹配的元件 |
| 操作方式 | 点击位号后，展示该器件的所有BOM原始参数，用户手动搜索并绑定 |
| 参数展示 | 将BOM上所有数据类信息完整罗列，方便比较参数 |
| 搜索范围 | 立创封装库（与自动匹配使用相同数据源） |
| 可修改性 | 手动匹配结果同样可以后续修改 |

### F6: 封装绑定

| 项目 | 说明 |
|------|------|
| 绑定操作 | 用户点击【绑定封装】按钮 |
| 绑定规则 | 只修改封装字段，器件除封装外的其他参数全部同步商城数据 |
| 批量绑定 | 支持选中多个器件后批量执行绑定 |
| 封装检索 | 能够自动检索立创封装库内所有封装 |
| C编号同步 | 匹配的器件能通过C编号找到立创器件，并支持随时同步修改 |
| 作用目标 | 将选中封装赋值给原理图中对应器件 |

### F7: DRC检查

| 项目 | 说明 |
|------|------|
| 检查时机 | 绑定前/绑定后均可触发 |
| 检查项1 | 原理图符号引脚数与封装焊盘数不匹配 → 警告提示 |
| 检查项2 | PN结类元件（二极管、三极管等）极性方向确认 → 提示用户核对 |
| 提示方式 | 在界面中高亮标记异常项，便于后续检查核对 |

### F8: 绑定结果导出

| 项目 | 说明 |
|------|------|
| 导出内容 | 原BOM数据 + 最新绑定的器件参数 |
| 自定义字段 | 用户可自行选择导出哪些BOM标题键值对 |
| 导出格式 | Excel / CSV |
| 字段来源 | 原始BOM字段 + 立创商城同步的器件参数字段，均可勾选 |

## 六、UI交互设计

### 页面1: BOM上传页

- 插件打开后首先展示此页面
- 包含文件上传区域（拖拽或点击上传）
- 支持的文件格式提示
- 上传后自动解析并跳转至数据展示页

### 页面2: BOM数据展示与分组页

- 以表格形式展示导入的BOM数据
- 相同参数的器件自动合并为一组
- 位号列收敛显示（展开/收起）
- 每行显示：组序号、代表型号、关键参数摘要、位号数量、匹配状态

### 页面3: 匹配依据选择

- 将BOM表中的所有数据类（列标题）提取并展示
- 每个数据类以标签/复选框形式呈现
- 用户可选择一个或多个作为匹配依据
- 选择后点击"开始匹配"触发自动匹配流程

### 页面4: 器件详情展示

- 点击任意位号/器件行时展开详情面板
- 完整罗列该器件在BOM上的所有原始数据（所有键值对）
- 以键值对列表形式展示，便于手动匹配时比较参数

### 页面5: 匹配结果页

展示匹配到的器件信息，包含：

- **封装3D/2D模型预览图** — 直观查看封装外形
- **原理图符号预览** — 查看IC引脚定义
- **器件完整参数** — 立创商城中该器件的所有参数信息
- **匹配置信度/来源** — 标注来自基础库/贴片库存/拓展库

### 页面6: 绑定确认页

- 【保存并绑定】按钮
- 封装检索功能：可搜索立创封装库内所有封装
- C编号关联：通过C编号定位立创器件，支持随时同步修改
- 绑定预览：展示绑定前后的对比

### 页面7: 导出配置页

- 列出所有可导出的字段（原BOM字段 + 新绑定参数字段）
- 复选框勾选需要导出的项
- 选择导出格式（Excel / CSV）
- 导出按钮

## 七、业务规则

### 7.1 匹配规则

1. 以用户选择的数据类字段值为搜索关键词；
2. 多个数据类同时选择时，取交集匹配（AND逻辑）；
3. 查询顺序：基础库 → 贴片库存 → 拓展库；
4. 优先返回完全匹配结果，无完全匹配时返回模糊匹配结果并标注。

### 7.2 绑定规则

1. 绑定时**只修改封装字段**；
2. 器件除封装外的其他参数（型号、描述、参数值等）**全部同步商城数据**；
3. 绑定目标为原理图中对应器件的封装属性。

### 7.3 分组规则

1. BOM中所有键值对参数完全一致的器件归为同一组；
2. 位号不参与分组判断；
3. 同组器件共享匹配结果，绑定时统一应用。

### 7.4 DRC规则

| 检查项 | 触发条件 | 提示级别 |
|--------|----------|----------|
| 引脚数不匹配 | 原理图符号引脚数 ≠ 封装焊盘数 | 错误（红色） |
| 极性未确认 | PN结类元件（二极管、三极管、电解电容等） | 警告（黄色） |
| 封装尺寸异常 | 封装尺寸与器件参数明显不符 | 警告（黄色） |

## 八、技术架构参考

本插件基于 EasyEDA Pro 扩展插件体系开发，架构参考现有 GenCAD Viewer 插件：

### 8.1 项目结构

```
eext-bom-footprint-matcher/
├── extension.json          # 扩展清单（菜单、入口、元数据）
├── package.json            # 项目配置与构建脚本
├── tsconfig.json           # TypeScript 配置
├── src/
│   └── index.ts            # 扩展入口（导出函数绑定菜单）
├── iframe/
│   └── main.html           # 主界面（BOM管理、匹配、绑定全流程UI）
├── config/
│   ├── esbuild.common.ts   # esbuild 公共配置（IIFE, browser, bundle）
│   └── esbuild.prod.ts     # 生产构建
├── build/
│   └── packaged.ts         # 打包为 .eext
└── .edaignore              # 打包排除规则
```

### 8.2 核心技术要点

| 要点 | 说明 |
|------|------|
| 入口模式 | `src/index.ts` 导出命名函数，通过 `extension.json` 的 `registerFn` 绑定菜单项 |
| UI承载 | 主要交互界面通过 `eda.sys_IFrame.openIFrame()` 打开 iframe 页面 |
| 宿主通信 | 通过全局 `eda` 对象调用 EasyEDA Pro API（sys_IFrame、sys_Dialog、sys_FileSystem 等） |
| 构建格式 | esbuild 输出 IIFE 格式，`globalName: 'edaEsbuildExportName'` |
| 数据请求 | iframe 内通过 HTTP 请求访问立创商城 API / LCEDA 封装库 API |
| 引擎兼容 | `engines.eda: "^3.0.0"` |

### 8.3 关键API使用

```typescript
// 打开主界面
eda.sys_IFrame.openIFrame('/iframe/main.html', width, height, 'bom-matcher', {
    title: 'BOM器件智能封装匹配',
    maximizeButton: true,
    minimizeButton: true,
});

// 文件选择（上传BOM）
eda.sys_FileSystem  // 文件打开对话框

// 消息通知
eda.sys_Message     // Toast 提示
eda.sys_Dialog      // 模态对话框（确认绑定、DRC警告等）
```

## 九、非功能性需求

### 9.1 性能要求

- 单个器件匹配响应时间 ≤ 2秒；
- 100个器件批量匹配 ≤ 30秒；
- UI操作（切换页面、展开详情）响应 ≤ 200ms。

### 9.2 兼容性

- EasyEDA Pro 版本 ≥ 3.0.0；
- BOM文件编码支持 UTF-8、GBK；
- 支持 Windows / macOS 平台。

### 9.3 错误处理

- 网络异常时提示用户检查网络连接；
- BOM解析失败时提示格式错误并给出支持格式说明；
- 匹配无结果时明确提示"未找到匹配项"，引导用户手动匹配；
- 绑定失败时回滚操作，不影响原有数据。

### 9.4 数据安全

- BOM数据仅在本地解析，不上传至第三方服务器（查询API除外）；
- 用户操作记录不持久化存储。

## 十、术语表

| 术语 | 说明 |
|------|------|
| BOM | Bill of Materials，物料清单 |
| 数据类 | BOM表中的列标题/字段名（如 Footprint、Value、Part Number） |
| 位号 | 元器件在原理图中的唯一标识（如 R1、C2、U3） |
| 封装 | 元器件的物理外形定义，用于PCB布局 |
| C编号 | 立创商城的器件唯一编号（如 C12345） |
| DRC | Design Rule Check，设计规则检查 |
| 基础库 | LCEDA 内置的基础封装库 |
| 贴片库存 | 立创商城有贴片库存的器件集合 |
| 拓展库 | LCEDA 社区/扩展封装库 |
