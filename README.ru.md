# GenCAD Viewer

[English](./README.en.md) | [简体中文](./README.md) | [繁體中文](./README.zh-Hant.md) | [日本語](./README.ja.md) | Русский

Расширение для EasyEDA Pro, позволяющее импортировать и просматривать файлы формата GenCAD (.cad).

## Возможности

- Добавляет пункт меню «Open GenCAD Viewer...» в редакторе печатных плат и на главной странице
- При нажатии открывает GenCAD Viewer в максимизированном iframe-диалоге
- Диалог поддерживает кнопки сворачивания/разворачивания
- Поддержка интернационализации (китайский и английский)

## Использование

1. Установите расширение в EasyEDA Pro
2. Откройте редактор печатных плат (или главную страницу), нажмите «GenCAD查看器» → «Open GenCAD Viewer...» в верхнем меню
3. В открывшемся диалоге загрузите файл `.cad` для просмотра
